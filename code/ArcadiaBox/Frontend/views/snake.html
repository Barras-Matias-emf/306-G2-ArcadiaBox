Snake

Comment jouer

Déplacez le serpent pour manger la nourriture, gagnez des points et évitez les murs ainsi que votre propre corps.

Clavier

  • ← ↑ → ↓ — Déplacer le serpent (flèches)
  • A (ou touche A) — Démarrer (si affiché Play)
  • R — Restart (si besoin)

Manette (Xbox)

  • D‑pad — Déplacer le serpent
  • Stick gauche — Déplacer (tilt fort)
  • A — Play / confirmer
  • Bouton Menu / Start — (non utilisé actuellement)

Conseils

  • Le bouton Play lance l'état READY ; appuyez ensuite sur une direction pour démarrer.
  • La manette est détectée automatiquement. Appuyez sur A ou déplacez le stick/D‑pad pour commencer.
  • Le jeu empêche les inversions directes (ex. gauche → droite immédiat).
